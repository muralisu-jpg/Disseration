Point-algebra definitions (each event has a START and END point):
- BEFORE: A ends before B starts.        - AFTER: A starts after B ends.
- IBEFORE: A ends exactly when B starts.  - IAFTER: A starts exactly when B ends.
- INCLUDES: A starts before B AND ends after B (A contains B).
- IS_INCLUDED: A starts after B AND ends before B (A inside B).
- BEGINS: same start, A ends first.        - BEGUN_BY: same start, B ends first.
- ENDS: same end, B starts first.          - ENDED_BY: same end, A starts first.
- SIMULTANEOUS: same start AND same end.   - IDENTITY: same event re-mentioned.
- DURING / DURING_INV: A holds throughout B (a state/time span).
